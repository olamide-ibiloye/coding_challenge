Python version -- 3.11

How to use
- User can merge as many files as they want
- Install version ^3.11
- Run the merge_csv.py file
- Enter the number of files to be merged
- Enter the unique file path for all the files
- Check the current working directory for the merged_file

Observations
- The file3 of the large files have 3 missing values 
(file1 -> 500000 entries, file2 -> 499997, file3 -> 500000)

Assumptions
- All the entries have unique ids
- All files have their first column as id column
- All files have headers
